packers/unpackers for visual novel file formats

BGI		bgi/ethornell
		script decode/encode
		use garbro to extract script files from data files
DO		d.o.
		pack only, use garbro for unpack
Willadv		adv/will v1
		pack .arc files (filename 12 chars, 1 byte per char)
		use garbro or exbelarc (asmodean) for unpack
